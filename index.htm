Welcome to the WLED Pixels Dice Tray Usermod web installer!

  1. Plug in your ESP to a USB port. We will install WLED v0.15.0-b4 with the Pixels Dice Tray Usermod to it.
  2. Currently this mod is only built for ESP32-S3 8MB QPSI boards and the TFT layout is specifically for the T-QT Pro
  3. Hit "Install" and select the correct COM port. No device found?
  4. Get WLED installed and connected in less than 3 minutes!
